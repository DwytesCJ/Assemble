# Assemble
This project contains the software designed to run an automated assembly line robotic arm. The main goal here is to make factory automation flexible and accessible to everyone, not just software engineers.Instead of hardcoding the arm to do just one specific job forever, this program introduces a super simple graphical user interface (GUI). Through this visual control panel, anyone on the factory floor can switch between different built-in operations on the fly.Whether you need the arm to pivot from a packing task to a sorting routine, you can make it happen instantly with a few clicks. It completely removes the headache of rewriting code or redeploying software every time your production line needs a change, keeping your uptime high and your workflow smooth.What this program doesSimplifies daily operations by replacing complex command-line tools with a clean, visual dashboard.Boosts factory flexibility by allowing a single robotic arm to handle multiple distinct manufacturing jobs.Saves time by letting operators swap out pre-configured tasks safely and instantly during live production.
